# MVP Scope

In scope:

- Product Source Mode: Same product / Replace product
- Model Source: Auto creator / One model / Model per account
- Reference video upload
- Product image upload
- Optional or account-specific model image upload
- Gemini reference video analysis
- OpenAI product analysis
- OpenAI avatar analysis when model image exists
- GPT Product Selling Plan and Prompt Composer
- fal Kling generation with native audio
- Local MP4 caching for browser playback
- Debug display for analysis, asset profiles, final prompt, and provider error

Out of scope for this launch pass:

- Separate external render composition
- Separate voiceover provider
- Manual Hook format selection
- Manual Setting format selection
- Reference preset UI
- User-entered benefits/CTA fields
- Retry/evaluation loop
