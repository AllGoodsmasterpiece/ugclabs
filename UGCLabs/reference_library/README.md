# Reference Library

Store downloaded reference videos here before they are analyzed into presets.

Recommended file naming:

```text
ref_001.mp4
ref_002.mp4
ref_003.mp4
```

Start simple:

```text
1 video = 1 representative folder
```

Later, analyzed files can sit next to the video:

```text
ref_001.mp4
ref_001.analysis.json
ref_001.preset.json
```

Use the folder that best represents the video's main purpose. A video can later be tagged with multiple concepts in metadata.
