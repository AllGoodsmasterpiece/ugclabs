# Open Questions

Current launch questions:

- What is the measured fal Kling v3 Pro cost per 5s, 10s, and 15s generation?
- How stable is product identity across repeated generations with the same product element?
- How stable is model identity across repeated generations with the same model element?
- Which prompt phrasing gives the best reference-video motion transfer when the video is analysis-only?
- How often does native audio match visible mouth movement well enough for UGC?
- Should Auto creator be allowed for customer-facing launch, or marked as draft-only because identity can vary?
- What minimum product image quality should be required before generation?
- Should model-per-account support saved account assets before launch, or is upload-per-run acceptable for MVP?
- Which optional advanced fields should be exposed first: product benefits, banned phrases, or CTA?

Closed for current MVP:

- The standard generation provider is fal Kling v3 Pro Image-to-Video.
- Reference video is not sent directly to the generation model.
- Model source remains user-selectable.
- Product Source Mode remains user-selectable.
- External render composition and separate voiceover are not part of the current runtime.
